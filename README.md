# Flappy Variant (2D Arcade Game)

A simple 2D arcade game inspired by classic tap-to-fly mechanics. The player must navigate through continuously spawning obstacles without collision.

## Gameplay Overview

The player controls a flying character that must pass through gaps between obstacles. Each successful pass increases the score.

- Tap/Key input applies upward force
- Obstacles spawn procedurally
- Collision ends the game
- Score increases per obstacle cleared

## Core Mechanics Implemented

- Physics-based jump mechanic
- Procedural obstacle spawning
- Collision detection system
- Score tracking
- Game over and restart system
